HTTP协议

· 2019-07-08 ·

一些特殊符号在url中有其他含义所以在接受到HTTP请求的时候要对url部分进行转义, url中以?开始表示查询字符串, 以/开头表示路径.... HTTP的格式: http请求: 请求行: 方法 url http版本 \r\n 请求头: Key: Value\r\n \r\n 请求体........

http响应: 响应行: 版本 状态码 状态描述\r\n 响应头: Key: Value\r\n \r\n 响应体

HTTP方法: get post put head delete //1.1和1.0 options trace connect //1.1 一般GET没有请求体参数在url中 POST的参数在请求体中

HTTP状态响应码:
```
1xx 状态信息     接收的请求正在处理
2xx 成功            请求正常处理完毕
3xx 重定向        需要额外操作
4xx 客户端错误 服务器无法处理
5xx 服务器错误 服务器处理出错
```
